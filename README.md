# Telegram Bot для каталога техники

Бот для просмотра PDF-инструкций и презентаций по технике.

## Команды
- `/start` - начать работу
- `/help` - помощь
- `/stats` - статистика
- `/catalog` - все категории

## Установка и запуск

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Создайте файл `config.py` с токеном бота
4. Запустите: `python bot.py`
